L’outil de stress et de performance de Lync Server 2013 inclut des outils qui simplifient la planification de la capacité de Lync Server 2013. L’outil de stress et de performance de Lync Server 2013 vous aide à effectuer les opérations suivantes :

  - Simplifiez la planification de votre matériel pour Lync Server 2013.

  - Vous offrir des connaissances et des meilleures pratiques en matière d’optimisation des performances.

  - Évaluez les performances de vos déploiements Lync Server 2013 prévus.
