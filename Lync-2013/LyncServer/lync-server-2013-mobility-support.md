_**Dernière modification de la rubrique :** 2012-06-29_

Avec la mobilité Lync Server 2013, les utilisateurs ont accès à la fonctionnalité Lync 2013 sur leurs appareils mobiles. Les appareils mobiles pris en charge sont les suivants :

  - Appareils Apple iOS pris en charge

  - Windows Phone

  - Android

  - Nokia

Pour plus d’informations sur les fonctionnalités de mobilité et la configuration requise, reportez-vous à la rubrique [planification de mobilité dans Lync Server 2013](lync-server-2013-planning-for-mobility.md) dans la documentation de planification.
