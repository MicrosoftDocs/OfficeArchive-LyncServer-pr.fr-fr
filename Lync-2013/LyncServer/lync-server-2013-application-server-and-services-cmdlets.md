Les applications serveur font référence aux programmes individuels qui s’exécutent sous Microsoft Lync Server 2013. Vous pouvez gérer ces applications à l’aide de cmdlets à partir de Lync Server Management Shell.

<div>

## <a name="application-server-and-services-cmdlets"></a>Cmdlets Server et services d’application

Utilisez les applets de commande suivantes pour gérer les applications Lync Server 2013.

**Serveur et services d’application**

  - <span></span>  
    [Set-CsApplicationServer](https://technet.microsoft.com/library/Gg398562(v=OCS.15))

<!-- end list -->

  - <span></span>  
    [Get-CsServerApplication](https://technet.microsoft.com/library/Gg425948(v=OCS.15))

  - <span></span>

  - <span></span>
