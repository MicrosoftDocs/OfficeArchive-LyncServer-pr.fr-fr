_**Dernière modification de la rubrique :** 2012-12-11_

Une stratégie de version de client est composée d’un ensemble de règles de stratégie de version de client. Ces règles définissent les actions qui doivent être effectuées lorsque les utilisateurs tentent de se connecter avec des clients et des versions de client spécifiques.
